# ShopFlow — Orders and Returns

## Cancelling an order

A customer can cancel an order from the orders page while it is in `pending` or
`paid` status. Once an order has shipped, it can no longer be cancelled and must
instead be returned after delivery.

## Returns

To return a delivered item, the customer requests a return from the order
detail page within the return window. Returned items must be unused and in their
original packaging. Once the warehouse receives and inspects the return, a refund
is issued to the original payment method.

## Refund processing time

Approved refunds are issued to the original payment method and typically appear
within 5 to 7 business days, depending on the card issuer.
